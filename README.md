# 两个小程序项目

基于微信小程序写的两个工作项目（名片小程序，官网小程序）
---

## 目录结构
- images — 存放项目图片
- pages — 存放项目页面相关文件
- style — 存放独立wxss样式文件，可import引入
- utils — 存放utils文件，可require引入
